    .NET

    Dynatrace OneAgent 通过在代码中的关键位置放置跟踪语句来检测 .NET 应用程序，以进行代码跟踪，性能指标，错误检测，依赖关系跟踪等。

    并非每个检测到的 .NET 应用程序都默认进行检测。Dynatrace 提供一组规则来检测特定进程（例如，IIS 应用程序池，您可以使用我们自己的规则进行扩展）。

    功能

    Dynatrace 提供广泛的 .NET 监控功能：

    • Web 请求服务，Web 服务，远程服务，队列和数据库的端到端事务跟踪。详细了解服务。
    • 垃圾收集，流程指标等等。
    • 不间断 24 x7 生产级 CPU 性能分析。
    • OneAgent SDK 用于自定义跟踪。

    有关支持的框架的详细信息，请参阅我们支持的技术矩阵。

    .NET 运行时

    Dynatrace 支持 .NET Framework 和 .NET Core。

    有关受支持的 .NET 版本的详细信息，请参阅我们支持的技术矩阵。

    支持生命周期

    不同的 CLR 根据其供应商提供不同的支持时间表。Dynatrace 根据其支持生命周期致力于支持每个版本。

    .NET Framework

    可以在 Microsoft 支持生命周期中找到 .NET Framework 支持策略。

    .NET 核心

    .NET Core 是开源的，但是由 Microsoft 正式支持。可以在 Microsoft 支持生命周期中找到支持策略。
