    支持 JVM

    Dynatrace 支持主要的 JVM 和 JDK。

    Oracle Hotspots JVM

    Oracle 支持矩阵

    JVM 版本6,7和8正在获得 Oracle 的无限期持续支持。Dynatrace 对这些版本的支持目前也是开放式的。

    版本9和10是非 LTS 版本，将在2018年结束。Dynatrace 将支持它们直到2019年底。版本11是 LTS ，其支持目前是开放式的。

    OpenJDK

    OpenJDK 是开源的，但 RedHat 正式支持。请参阅支持生命周期。

    Dynatraces 目前为版本6,7和8提供开放式支持。

    版本9和10是非 LTS 版本。Dynatrace 将支持这些直到 2019 年底。版本11是 LTS，其支持目前是开放式的。

    SAP JVM

    SAP JVM 是由 SAP 构建和维护的 JVM，基于 OpenJDK 。查看详细信息和官方版本。

    SAP JVM 6 于 2017 年底停产，不再可用。Dynatrace 目前为版本7和8提供开放式支持。

    Java 版本 供应商发布 供应商 EOL 由 Dynatrace 支持，直到 最后支持的 OneAgent 版本
    6 2017 年 12 月 2018 年 12 月 1.157
    7 不定
    8（LTS） 不定

    IBM JVM

    IBM JVM 有自己的支持时间表，但实际支持基于对其他 IBM 产品的支持。对于 Dynatrace，此处最重要的产品是 WebSphere Application Server（请参阅支持时间表）。

    WebSphere Application Server 处于增强型生命周期策略中。这意味着尽管 WebSphere 8 是 2018 年 4 月的 EOS，但 IBM 将继续支持它直到 2021 年。

    只要 IBM 支持，Dynatrace 就会支持 WebSphere 和底层 JVM 版本。有关详细信息，请参阅支持的技术矩。

    这意味着，目前，IBM JVM 6,7和8具有开放式支持。

    Java 版本 供应商发布 供应商结束支持 由 Dynatrace 支持，直到 最后支持的 OneAgent 版本
    6 2007 年 11 月 2017 年 9 月 不定
    7 2011 年 9 月 2019 年 9 月 不定
    8（LTS） 2015 年 2 月 2022 年 4 月 不定

    注意：

    内存转储需要 IBM JVM 7 或更高版本。 了解有关 IBM JVM 的已知问题和解决方案的更多信息。

    支持与否

    JVM 根据其供应商提供不同的支持时间表。Dynatrace 致力于支持每个 JVM 和版本至少与其供应商一样长，并且在大多数情况下至少延长1年。

    有关受支持的 JVM 的更多详细信息，请参阅我们支持的技术矩阵。
