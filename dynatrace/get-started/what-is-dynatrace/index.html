    什么是 Dynatrace？

    Dynatrace 是一个软件智能监控平台，可简化企业云复杂性并加速数字化转型。通过 AI 和完全自动化，Dynatrace 一体化平台提供有关应用程序性能，底层基础架构和最终用户体验的答案，而不仅仅是数据。Dynatrace 用于实现企业云操作的现代化和自动化，更快地发布更高质量的软件，并为组织的客户提供最佳的数字体验。

    Dynatrace 将基础架构和云，应用程序性能和数字体验监控无缝地集成到由人工智能驱动的一体化自动化解决方案中。Dynatrace 通过为开发，运营和业务团队提供共享平台指标，帮助提高绩效结果。通过这种方式，Dynatrace 可以作为您组织的单一“事实来源”。

    Dynatrace功能

    Dynatrace 提供以下功能，用于监视和分析应用程序环境各个方面的性能。

    • 真实用户监控

    无论是在浏览器中还是在移动设备上进行交互，真实用户监控都会分析所有用户与您的应用程序交互的性能。真实用户监控还支持应用程序可用性监控，UI 元素正确显示验证，第三方内容提供商性能分析，后端服务性能分析（深入至代码级别）以及所有底层基础架构的性能分析。

    • 移动应用监控

      Dynatrace 还支持移动应用程序的真实用户监控。监控本机移动应用程序的用户体验的过程与监视基于浏览器的Web应用程序有根本的不同。这是因为移动应用程序监控涉及监控库的编译，打包和装运以及您自己的移动应用程序包。仪表移动应用程序的过程在很大程度上取决于移动应用程序的平台。Dynatrace 支持 Android 和 iOS 平台。

    • 服务器端服务监控

      Web 应用程序由 Web 服务器（例如，Apache Tomcat）和 Web 容器（例如，Docker）提供的Web页面组成。发送到特定 Tomcat 服务器的 Web 请求是服务器端服务的示例。服务器端服务可以是各种类型，如 Web 服务，Web 容器，数据库请求和自定义服务。Dynatrace OneAgent 可以提供有关哪些应用程序或服务与特定服务调用的其他服务以及哪些服务或数据库交互的详细信息。

    • 网络，进程和主机监控

      Dynatrace 支持监控整个基础架构，包括主机，进程和网络。您可以执行日志分析并查看信息，例如网络的总流量，主机的 CPU 使用率，进程的响应时间等。Dynatrace 还提供详细的拓扑信息，以便您了解哪些进程在哪些主机上运行以及您的进程如何互连。

    • 云和虚拟机监控

      Dynatrace OneAgent 可监控整个堆栈，包括私有云，公共云和混合云环境。无论您是在 AWS，Azure，Cloud Foundry还是 OpenStack 上运行，Dynatrace OneAgent 都会自动检测所有虚拟化组件并跟上所有更改。Dynatrace OneAgent 可以与您的虚拟化基础架构集成，允许您在数据中心的 vCenters 依赖关系，运行它们的进程和应用程序之间连接点。

    • 容器监控

      Dynatrace 与现有 Docker 环境无缝集成，可自动监控您的容器化应用程序和服务。Dynatrace 挂钩到容器中，并提供将 OneAgent 注入容器化进程的代码。

      无需修改 Docker 镜像，修改运行命令或创建其他容器来启用 Docker 监控。只需在为容器化应用程序和服务提供服务的主机上安装 Dynatrace OneAgent。Dynatrace 自动检测容器的创建和终止，并监视这些容器中包含的应用程序和服务。

    • 根因分析

      Dynatrace 的一个关键特性是戴维斯，即 Dynatrace AI 驱动的因果引擎。Davis 依靠人工智能来持续监控应用程序，服务和基础架构的各个方面，以自动学习所有这些组件的基准性能指标和依赖关系。Dynatrace 还会自动了解对您的业务成功至关重要的关键请求的基线响应时间和故障率（客户新通知注册，客户将项目添加到在线购物车，请求现场演示等），以便您可以跟踪成功实现您的业务转换目标。问题检测和报告基于这些基线值。例如，当检测到的减速或错误率增加证明生成新问题事件时，Dynatrace 确定。

    Dynatrace 如何提供其功能

    Dynatrace 独有的三项专利技术相互吻合，可以自动发现，建模和分析每个组件以及应用程序所有层之间的依赖关系。Dynatrace 提供真正的全栈监控。

    OneAgent 技术使用单个代理来收集和统一应用程序环境中所有类型实体的所有操作和业务性能指标 - 服务器，应用程序，服务，数据库等 - 跨技术堆栈的每一层（包括容器！）。

    Smartscape 可视化技术可映射环境中运行的所有内容，并检测网站，应用程序，服务，进程，主机，网络和云基础架构之间的所有因果依赖关系。

    Dynatrace 获得专利的 PurePath 技术可捕获所有支持技术（从云到大型机）端到端应用程序事务的时序和代码级上下文。

    Dynatrace 部署模型

    Dynatrace 可以作为 SaaS 解决方案部署，也可以部署在本地部署中。本地版本称为 Dynatrace Managed。

    SaaS 客户只需要安装 OneAgent。在安装 OneAgent 之前，托管客户需要设置他们的 Dynatrace 集群。

    对于这两种部署模型，强烈建议您安装适当类型的 ActiveGate。ActiveGate 提供了许多特定于代理的功能。
